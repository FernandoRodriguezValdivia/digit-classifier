## 🎯 ¿Qué hace?

El usuario dibuja un dígito (0-9) en un canvas y una **red neuronal convolucional (CNN)** entrenada con el dataset **MNIST** predice qué número es, mostrando el nivel de confianza de la predicción.

**Todo corre 100% en el navegador. No hay backend. No se envían datos a ningún servidor.**
